## 객체

- {} 중괄호 사용
- 키, 값 쌍으로 프로퍼티를 여러개 넣을 수 있다.
- 키에는 문자형, 값에는 모든 자료형

<br>

## 객체 생성

```jsx
let user = new Object(); // '객체 생성자' 문법
let user = {}; // '객체 리터럴' 문법
```

<br>

## 프로퍼트 값 얻기

```jsx
// 프로퍼티 값 얻기
alert(user.name); // John
alert(user.age); // 30
```

<br>

## 프로퍼티 값엔 모든 자료형이 올 수 있습니다

```jsx
user.isAdmin = true;
```

<br>

## 프로퍼티 삭제하기

```jsx
delete user.age;
```
